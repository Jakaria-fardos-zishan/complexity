\section{\label{sec:level1}Introduction}

The Lorenz attractor, an example of deterministic chaos, traces a beautiful butterfly-shaped trajectory. By simply observing this trajectory, we instantly recognize its inherent complexity. Yet quantifying this complexity remains profoundly difficult. We can compute existing complexity measures, but this immediately raises a vital question: 'Are these measures even appropriate for capturing the true complexity of this system? If not, what alternative approach can we discover?'

If we succeed in establishing a meaningful new quantity for measuring complexity, it must carry genuine physical significance. Simultaneously, we can explore entropies that accurately describe the system. With so many entropy definitions available, we must ask: Do all entropies tell the same story, or do they reveal fundamentally different aspects? Moreover, we could seek novel kinds of entropy that might describe this system more effectively.

Given its chaotic nature, we might also calculate quantities like Lyapunov exponents, multifractal spectra, etc. Crucially, we have three time series (x, y, z). Each can be studied individually, but we must understand their interdependence. Therefore, these series demand mutual analysis.

To uncover relationships between key quantities, we can employ tools like autocorrelation, mutual information, transfer mutual information, etc. Beyond dynamics, we may also probe geometric properties â€“ extracted not only from the trajectory itself, but also from phase space, velocity space, and beyond.

\begin{verbatim}
\appendix

\end{verbatim}
will produce an appendix heading that says ``APPENDIX A: BACKGROUND''
(note that the colon is set automatically).

If there is only one appendix, then the letter ``A'' should not
appear. This is suppressed by using the star version of the appendix
command (\verb+\appendix*+ in the place of \verb+\appendix+).
